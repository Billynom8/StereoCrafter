Compiling Forward Warp can be tricky so here is a fix:

*NOTE* This version of Forward Warp was built for Python == 3.12

- Copy all folder contents to StereoCrafter environment "site-packages" folder

Example PATH for anaconda:
C:\Users\[username]\anaconda3\envs\StereoCrafter\Lib\site-packages

Example PATH for Python virtual environment (venv):
D:\StereoCrafter\.venv\Lib\site-packages
